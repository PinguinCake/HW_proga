Лабораторная работа: Пошаговая консольная стратегия на Java
------------------------------------------------------------

Автор: Гуляева Юлия
Группа: ИУ3-41Б

Описание:
Реализована игра в духе Heroes of Might and Magic:
- Пошаговая стратегия
- Режим PvP (2 игрока за одной консолью)
- Режим PvE (игрок против ИИ)
- Система замков, армий, зданий, золота
- Боевая система с отдельным полем боя
- Победа по захвату замка или уничтожению героя

Команды:
- move X Y — переместиться на клетку
- attack — начать бой, если на клетке есть враг
- buy — купить юнитов
- build — построить здание
- stats — посмотреть информацию о герое (HP, армия, золото, здания)
- endturn — завершить ход
- help — показать список команд
- exit — выйти из игры

Как запустить:
1. Открыть проект в IntelliJ IDEA
2. Открыть файл Main.java
3. Запустить main()
4. Следовать инструкциям на экране

Удачной игры!
